# 📖 [Bookstore] <a name="about-project"></a>

**[Bookstore]** is a website that allows you to: display a list of books, add a book, and remove a selected book.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to have installed Node JS

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/AlejandraKeber/bookstore.git
```

### Install

Install this project with:

```sh
  cd bookstore
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```
